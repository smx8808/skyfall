Configure Our Routers

R1 and R2 are the Cisco virtual routers that will be used for the site-to-site IPSec VPN. We will manually configure the tunnel and endpoints, so this will be a static virtual tunnel interface. Computer 1 and Computer 2 are being used to test connectivity and the tunnel.

Step 1

Configure Router 1 – R1:

R1 (Router 1)

Network Configuration
R1(config)#int Gi0/2
R1(config)#ip address 192.168.10.1 255.255.255.0
R1(config)#no shut
R1(config)#int Gi0/1
R1(config)#ip address 192.168.100.254 255.255.255.0
R1(config)#no shut

IPSec Phase 1 Configuration:
R1(config)#crypto isakmp policy 1
R1(config-isakmp)#encryption aes
R1(config-isakmp)#authentication pre-share
R1(config-isakmp)#group 2

Configure remote neighbor address (R2):

R1(config-isakmp)#crypto isakmp key SECRET address 192.168.10.2

Configure phase 2:

R1(config)#crypto ipsec transform-set TRANSFORM_SET esp-aes esp-sha-hmac 
R1(cfg-crypto-trans)#mode tunnel
R1(config)#crypto ipsec profile IPSEC_PROFILE
R1(ipsec-profile)#set transform-set TRANSFORM_SET

Combine everything on the tunnel interface:

R1(config)#interface Tunnel 0
R1(config-if)#ip address 1.1.1.1 255.255.255.252
R1(config-if)#tunnel source 192.168.10.1
R1(config-if)#tunnel destination 192.168.10.2
R1(config-if)#tunnel mode ipsec ipv4
R1(config-if)#tunnel protection ipsec profile IPSEC_PROFILE

The configuration of the tunnel interface is similar to a regular GRE tunnel. We set a source and destination IP address. The tunnel mode, however with IPSec IPv4 we have to add our IPSec profile.

Configure route that points to the subnet on the other side. The destination is the tunnel interface:

R1(config)#ip route 192.168.2.0 255.255.255.0 Tunnel0

R2 (Router 2)

The configuration of router R2 is nearly identical except for the IP addresses:

Network Configuration

R2(config)#int Gi0/2
R2(config)#ip address 192.168.10.2 255.255.255.0
R2(config)#no shut
R2(config)#int Gi0/1
R2(config)#ip address 192.168.200.254 255.255.255.0
R2(config)#no shut
R2(config)#crypto isakmp policy 1
R2(config-isakmp)# encryption aes
R2(config-isakmp)# authentication pre-share
R2(config-isakmp)# group 2
R2(config-isakmp)#crypto isakmp key SECRET address 192.168.10.1 
R2(config)#crypto ipsec transform-set TRANSFORM_SET esp-aes esp-sha-hmac 
R2(cfg-crypto-trans)# mode tunnel
R2(config)#crypto ipsec profile IPSEC_PROFILE
R2(ipsec-profile)# set transform-set TRANSFORM_SET 
R2(config)#interface Tunnel0
R2(config-if)# ip address 1.1.1.2 255.255.255.252
R2(config-if)# tunnel source 192.168.10.2
R2(config-if)# tunnel destination 192.168.10.1
R2(config-if)# tunnel mode ipsec ipv4
R2(config-if)# tunnel protection ipsec profile IPSEC_PROFILE
R2(config)#ip route 192.168.1.0 255.255.255.0 Tunnel0

Be sure to save your configuration: wr mem or copy run start
